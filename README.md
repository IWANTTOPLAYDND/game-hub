# 🎮 Arcade OS (Pixel Arcade Hub)

A browser-based arcade hub built with HTML, CSS, and JavaScript.  
It contains multiple mini-games, a lightweight game engine system, and save/progression features.

---

## 🕹️ Features

- 🎮 Multiple built-in mini games
  - Clicker
  - Reaction Timer
  - Number Guessing
  - Tic Tac Toe
  - Snake (classic-style)
  - and more to come
- 💾 Local save system (scores + progress)
- 🎨 Theme system (pixel / neon / glitch styles)
- 🔊 Sound effects engine
- 📈 XP and level system
- 🧠 Modular JavaScript structure (easy to expand)
